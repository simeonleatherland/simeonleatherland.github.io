Projects

Maps

Part of Data Structures and Algorithms Unit it is a Java based program that allows users to import a preset list of locations and distances between them. The program creates a virtual graph of this information utilizing multiple different data structures to allow the user to then select two locations for the program to generate the fastest time depending on traffic conditions/mode of trasnport and peak time configerations. This program generates the fastest time via Dijkstra's algroithm and a brute force method (which makes it really bad).

Spell Checker

This standalone spell checker allows a text file to be checked against a local dictionary. It can either automatically change spelling or ask the user if they wish to update the word. This is done in C89.

Trekking

Trekking is a command based application that takes in multiple routes and stores them, allowing the user to then follow along a specific trail and, given the users location, update them on where they are on the trail and their next destination. This was done using many software design patterns taught in Object Orientated Software Engineering.
